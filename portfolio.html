IMRS

For my senior capstone project, two teams of UNT students are working together to create the Incident Monitoring and Response System (IMRS), with the sponsorship of A-Shield, LLC. IMRS is a system that will allow students, teachers, and staff in active shooter situations to quickly provide information to law enforcement as to the whereabouts of the shooter. Currently, this system is in the design phase, but will use AWS infrastructure to get location and status information from students, teachers, and staff to law enforcement.

Password Manager

For my Foundations of Cybersecurity class, the end of the semester project that we worked on allowed us to choose between several project types. I chose to make a simple password manager. The program was written in Java, and I used the JavaFX framework to create the GUI. User passwords were encrypted using AES-256, and the program required the entry of a key unique to the user in order to view the encrypted passwords.
